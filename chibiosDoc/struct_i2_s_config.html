int16_t I2SConfig::i2scfgr

Configuration of the I2SCFGR register.

See the STM32 reference manual, this register is used for the I2S configuration, the following bits must not be specified because handled directly by the driver:

  • I2SMOD
  • I2SE
  • I2SCFG
